# Task Tracker Application

## Technology Used
HTML, CSS, JavaScript, ExpressJS

## Features Completed
- Add task
- View task list
- Mark task as completed
- Delete task
- activityHistory (api as of now)

## How to Run
- Using Live Server or Open .html file
- node app.js (To run express server) or using nodemon

## Challenges Faced
 - DOM Manipulation Of Task Components
 - Handeling states of Tasks
